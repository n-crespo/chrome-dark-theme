Install here: [Chrome Web Store](https://chromewebstore.google.com/detail/jicmcjfnodkkpbpiabkflcmjjapeacmd?utm_source=item-share-cb)

Or install manually:

1. [Download zipped source code](https://github.com/n-crespo/chrome-dark-theme/archive/refs/heads/main.zip) and unzip locally
2. visit <chrome://extensions> and enable developer mode
3. click "Load unpacked"
4. select the folder you just unzipped

A simple matte black and dark grey theme for Google Chrome/Chromium/Helium browser.
